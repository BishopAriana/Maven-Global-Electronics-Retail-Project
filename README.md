# Data Model & Interactive Revenue Report in Excel

## Problem Outline
Maven Electronics, a global electronics retailer (online and in-store) has had a decline in revenue since 2020. Management needs help consolidating the data to conduct an exploratory analysis.

## Methodology
The provided CSV files were used to build a data model and interactive report in Excel.

## Skills
Excel, Data Modeling, Power Pivot, Pivot Tables, Excel Charts, Slicers, KPI Cards
